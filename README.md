# 🥤 Refri do Dev

Projeto de um **slider animado de refrigerantes** desenvolvido com **HTML, CSS e JavaScript puro**, inspirado em um tutorial do YouTube do canal **DevClub**.

📺 Vídeo utilizado como base:
[https://www.youtube.com/watch?v=QLbrpjfltFs&t=772s](https://www.youtube.com/watch?v=QLbrpjfltFs&t=772s)
Canal: **DevClub**

---

## 🚀 Demonstração

O projeto exibe:

* 🍓 Morango
* 🥑 Abacate
* 🍊 Laranja

Com navegação por setas (`<` e `>`), alternando entre os sabores com animação.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** → Estrutura da página
* **CSS3** → Estilização e animações
* **JavaScript (Vanilla JS)** → Controle da troca de slides

---

## 🎯 Funcionalidades

✅ Slider com troca de itens
✅ Mudança dinâmica da cor de fundo
✅ Animação ao ativar um novo item
✅ Navegação circular (loop infinito)
✅ Estrutura simples e organizada

## 📸 Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/elivaldobraga23/refri_do_dev.git
```

2. Abra o arquivo `index.html` no navegador.

Ou utilize a extensão **Live Server** no VS Code para melhor experiência.

Neste projeto pratiquei:

* Manipulação de DOM
* Uso de `classList`
* Controle de estado com JavaScript
